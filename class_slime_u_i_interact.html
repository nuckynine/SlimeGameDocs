Detailed Description

A script that is used for the slime to interact with the slime GUI. Detects when the player is close enough to activate the GUI gameobject, and acts accordingly when the player moves away or wishes to exit the GUI. While the GUI is active, the script will keep updating the GUI until the GUI is deactivated.
