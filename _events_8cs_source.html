4//This class/script handles all the custom events.
5//Events make funtions that span multiple scripts much easier to implement.
6//It also reduces the amount of references between scripts, which makes updating/deleting scripts that interact with one another much easier.
7//What they do is, they enable an entire set of functions, from different scripts to be executed all at once.
8//All you have to do is execute the event, then all functions subscribed to it will execute.
9
10//The way to do it is:
11
12//1. Declare an event Action: "public event Action example". Depending on the usage of it, you can also assign a return type: "public event Action<float> example"
13/*
14 public event Action exampleEvent; Note: the event's return type has to match the function it is subsribing to.
15 public event Action<float> exampleEventFloat; <== Example of an event with return type float.
16
17 //If the return type is ommited, the event considers it as a void return type.
18*/
19
20//2. Encase the event in a public void, so that when calling the event, you call it through the function.
21// Within the encasing, check if the event is not null. If it isnt, execute the event.
22// This is because if the event were called while there was no function assigned, it will return an error.
23/*
24 public event Action<float> exampleEventFloat
25
26 public void ExampleEventFloat(float returnFloat){
27 if(exampleEventFloat != null){
28 exampleEventFloat(returnFloat);
29 }
30 }
31*/
32
33//3. Go to the script which has the function you want this event to execute, and create a reference to this script there.
34/*
35 [SerializeField]private Events events; <== SerializeField is a good practice to keep variables private, while displaying in the editor.
36*/
37
38//4. In the Start/enable method there, simply add the desired function to the event:
39/*
40 void Start(){
41 events.exampleEvent += ExampleFunction; Note that there is no parenthesis() when adding a function
42 events.exampleEventFloat += ExampleFunctionFloat;
43 . . .
44 }
45*/
46//This subsribes the function to the event
47//Now, any script that has a reference to this events script, can call those functions subscribed to the event.
48/*
49 events.ExampleEvent(); <== For void return type
50 events.ExampleEventFloat(someFloat); <== For other return types
51*/
